# Public Ethics of Work

Public Ethics of Work is an open-source app for logging work as a public ethical practice.

It is not a productivity tracker.
It is a structured work log that makes visible:

- achievements
- fuckups
- regen
- repair
- projects
- reflections
- balance

The app is designed for people whose work is fragmented, collective, moral, creative, pedagogical, political, and hard to reduce to output metrics.

## Core ideas

- work should be documented in public without pretending to be frictionless
- regeneration is part of work
- invisible work matters
- failures and dilemmas matter
- public reflection can be part of ethical practice
- the system should stay simple enough for daily use

## Main features (v1)

- quick daily logging
- public/private visibility
- internal project assignment
- public monthly log
- fuckup entries with optional public reflections
- recovery vs workload balance
- skin-based design system
- CZ/EN interface
