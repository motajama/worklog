# Contributing

Thanks for contributing to Public Ethics of Work.

## Project principles

This project is not a generic productivity app.
It is a framework for documenting work as an ethical, social, and political practice.

Please keep contributions aligned with these ideas:

- regeneration is part of work
- invisible work matters
- fuckups and dilemmas matter
- public reflection should be moderated
- simplicity matters more than feature bloat
- design should stay readable, raw, and non-corporate

## Areas for contribution

- bug fixes
- accessibility
- skins
- localization
- balance model improvements
- moderation tools
- documentation

## Before contributing

- open an issue for larger changes
- keep the code readable
- avoid unnecessary dependencies
- respect the tone and purpose of the project
